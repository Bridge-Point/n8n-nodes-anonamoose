# Changelog

## 0.1.0

Initial release.

- **Redact** — detect and replace PII using a four-layer pipeline (Dictionary, NER, Regex, Names)
- **Rehydrate** — restore original values using a session ID
- **Dictionary Add** — add terms for guaranteed redaction
- **Dictionary Delete** — remove terms by name
- Configurable locale support (AU, NZ, UK, US)
- Usable as an AI tool in n8n agent workflows
